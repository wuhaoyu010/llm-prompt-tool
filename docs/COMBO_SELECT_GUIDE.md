# 组合选择器使用说明

## 功能特性

### 1. 动态加载模型列表
- 自动从 `/api/models` 接口加载可用模型
- 支持从数据库读取默认模型配置
- 如果数据库中没有配置,默认选择第一个模型

### 2. 下拉选择
- 点击输入框或下拉按钮展开选项列表
- 显示模型名称和所有者信息
- 支持高亮当前选中项

### 3. 搜索过滤
- 在下拉框的搜索框中输入关键词
- 支持搜索模型 ID、名称和所有者
- 实时过滤显示匹配结果

### 4. 自定义编辑
- 可以直接在输入框中输入自定义模型名称
- 输入内容会实时过滤下拉选项
- 支持输入列表中不存在的模型名称

### 5. 键盘支持
- `Enter`: 选择第一个匹配项
- `Escape`: 关闭下拉框
- 其他按键: 正常输入

## 使用步骤

### 启动应用

```bash
# 方式1: 双击启动脚本
start.bat

# 方式2: 使用 conda 环境
conda activate prompt-tool
python run_standalone.py

# 方式3: 直接运行
F:\Anaconda\envs\prompt-tool\python.exe run_standalone.py
```

### 配置 API Key

1. 打开浏览器访问 http://localhost:5001
2. 点击右上角的设置图标 ⚙️
3. 在"大模型配置"标签页中:
   - 输入 API Key
   - 输入 API URL (默认: https://api.siliconflow.cn)
   - 选择或输入默认模型
   - 设置 Temperature 和 Max Tokens
4. 点击"保存设置"

### 使用模型选择器

#### 系统设置中的默认模型选择器

1. 打开系统设置 (点击右上角 ⚙️ 图标)
2. 切换到"大模型配置"标签
3. 点击"默认模型"输入框:
   - 下拉选择: 点击选项直接选择
   - 搜索: 在搜索框输入关键词过滤
   - 自定义: 直接输入模型名称
4. 保存设置

#### 实时推理对比处的模型选择器

1. 选择一个缺陷类别
2. 选择或创建一个测试用例
3. 在"实时推理对比"模块:
   - 点击模型选择器
   - 从下拉列表选择模型,或
   - 搜索过滤,或
   - 直接输入模型名称
4. 点击"运行对比"开始推理

## 常见问题

### Q: 模型选择器显示"请先配置 API Key"?
**A**: 这是因为没有配置 API Key。请先在系统设置中配置 API Key,然后刷新页面。

### Q: 下拉列表为空?
**A**: 可能的原因:
1. API Key 无效或已过期
2. API URL 配置错误
3. 网络连接问题
4. API 服务暂时不可用

解决方法:
1. 检查 API Key 是否正确
2. 检查 API URL 是否可访问
3. 检查网络连接
4. 查看浏览器控制台的错误信息

### Q: 如何输入自定义模型名称?
**A**: 直接在输入框中输入即可。组合选择器支持两种模式:
1. 从列表中选择(点击下拉选项)
2. 自定义输入(直接在输入框输入)

### Q: 搜索不区分大小写吗?
**A**: 是的,搜索会匹配模型 ID、名称和所有者,不区分大小写。

### Q: 如何验证模型选择是否生效?
**A**: 
1. 在系统设置中选择模型并保存
2. 刷新页面,查看是否显示之前选择的模型
3. 在实时推理对比中选择模型并运行测试
4. 检查后端日志,确认使用的模型名称

## 技术细节

### API 接口

**GET /api/models**

返回示例:
```json
{
  "models": [
    {
      "id": "Pro/Qwen/Qwen2.5-VL-7B-Instruct",
      "name": "Pro/Qwen/Qwen2.5-VL-7B-Instruct",
      "owned_by": "Qwen"
    }
  ],
  "default_model": "Pro/Qwen/Qwen2.5-VL-7B-Instruct"
}
```

### 数据流程

1. 页面加载时调用 `App.init()`
2. `initModelSelector()` 初始化组合选择器
3. 调用 `/api/models` 获取模型列表
4. 后端从数据库读取配置,返回模型列表和默认模型
5. 前端设置模型列表和默认值
6. 用户可以搜索、选择或自定义输入
7. 保存时将选择的模型发送到后端

### 组合选择器组件

```javascript
class ComboSelect {
    // 主要方法
    create()           // 创建 DOM 元素
    setModels(models)  // 设置模型列表
    setValue(value)    // 设置当前值
    getValue()         // 获取当前值
    setDisabled(bool)  // 启用/禁用
}
```
